# Screenshots (Platzhalter)

Hier die Screenshots der laufenden App ablegen, damit die READMEs sie anzeigen:

| Datei | Inhalt |
|---|---|

`http://localhost:8501`, jeweilige Ansicht öffnen und Screenshot speichern
(empfohlen: Breite ~1600 px, PNG).
